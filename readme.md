# kbframe-assist

kbframe框架的辅助模块

### 框架介绍
这是一款基于kbframe框架的辅助模块，用于生成迁移文件，填充数据，数据字典等辅助功能

### 安装
```php
$ cp .env.example .env //复制env文件，配置好相应的配置信息，数据库必须配置
```
```php
$ composer update
```
```php
$ php artisan key:generate
```
